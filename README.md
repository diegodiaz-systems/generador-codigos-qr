# Generador de Documentos PDF con Código QR

Aplicación desarrollada en Python que permite generar documentos PDF de forma automática
a partir de datos administrativos, integrando un código QR con información adicional
y recomendaciones para el usuario final.

Este proyecto fue desarrollado como parte de un proyecto académico con enfoque profesional,
utilizando datos ficticios y sin referencia a sistemas o información confidencial.

---

## Funcionalidades
- Captura de datos básicos (nombre del usuario y motivo administrativo)
- Generación automática de documento PDF
- Creación e integración de código QR dentro del PDF
- Visualización de información y recomendaciones al escanear el QR

---

## Tecnologías utilizadas
- Python 3
- Generación de PDFs
- Códigos QR
- Manejo de archivos

---

## Flujo de funcionamiento
1. El usuario ingresa los datos solicitados.
2. El sistema genera un documento PDF con un resumen del caso.
3. Se crea un código QR con información embebida.
4. Al escanear el QR, el usuario puede consultar detalles y recomendaciones.
